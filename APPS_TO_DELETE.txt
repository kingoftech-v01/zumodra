## üìù Removal Instructions

### Step 1: Backup
```bash
# Create backup before deletion
git add .
git commit -m "Backup before app cleanup"
```

### Step 2: Remove from INSTALLED_APPS
Edit `zumodra/settings.py` and remove:
```python
# REMOVE THESE FROM INSTALLED_APPS:
# 'dashboard_alert'
# 'dashboard_job'
# 'dashboard_project'
```

### Step 3: Remove Apps
```bash
# Windows (PowerShell)
Remove-Item -Recurse -Force jobs
Remove-Item -Recurse -Force projects
Remove-Item -Recurse -Force dashboard_alert
Remove-Item -Recurse -Force dashboard_job
Remove-Item -Recurse -Force dashboard_project

# Linux/Mac
rm -rf jobs/
rm -rf projects/
rm -rf dashboard_alert/
rm -rf dashboard_job/
rm -rf dashboard_project/
```

### Step 4: Clean Migrations
```bash
# After removing apps, create new migration to clean up
python manage.py makemigrations
python manage.py migrate
```

### Step 5: Verify
```bash
# Check for any remaining references
grep -r "dashboard_alert" .
grep -r "dashboard_job" .
grep -r "dashboard_project" .
grep -r "jobs\." .
grep -r "projects\." .
```

### Step 6: Commit
```bash
git add .
git commit -m "Remove empty and duplicate apps

- Removed jobs/ (duplicate of configurations.Job)
- Removed projects/ (duplicate of services models)
- Removed dashboard_alert, dashboard_job, dashboard_project (empty placeholders)
- Cleaned up INSTALLED_APPS in settings.py
"
```


### 5. Password Security

- Minimum 12 characters
- Mixed case required
- Number required
- Special character required
- No username in password
- No sequential characters (abcd, 1234)
- No repeating characters (aaaa)
- HIBP breach check (optional)
- Password history (last 12)

### 6. Session Security

- Redis-backed sessions
- 2-week expiry
- HTTP-only cookies
- Secure flag in production
- SameSite=Lax

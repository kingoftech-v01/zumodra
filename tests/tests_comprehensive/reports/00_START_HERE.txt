================================================================================
THE 7 TEST ASPECTS
================================================================================

1. PASSWORD RESET REQUEST (Email Sending)
   - Email sent on request
   - Reset link in email
   - No sensitive data exposed
   - CSRF protected form
   - Email enumeration prevented

2. RESET TOKEN GENERATION & VALIDATION
   - Cryptographically secure (HMAC-SHA256)
   - 2^256 entropy (unbreakable)
   - User-specific tokens
   - Proper signature verification
   - Token format validation

3. TOKEN EXPIRATION (Time-Limited)
   - 24-hour timeout configured
   - Expiration enforced
   - Clear error messages
   - User can request new token
   - Complies with OWASP guidelines

4. PASSWORD STRENGTH REQUIREMENTS
   - Minimum 8 characters
   - Mixed case required
   - Numbers and symbols
   - No common passwords
   - 4 validators enabled

5. PASSWORD CHANGE CONFIRMATION
   - Old password invalidated
   - New password works for login
   - PBKDF2-SHA256 hashing
   - 600,000 iterations
   - Sessions updated

6. ACCOUNT LOCKOUT (Brute Force Protection)
   - 5 failed attempts = lockout
   - 1-hour cooloff period
   - Per-IP tracking (axes)
   - Clear error messages
   - Manual admin unlock available

7. NOTIFICATION (Password Change Alert)
   - Email sent on password change
   - Timestamp included
   - IP address logged
   - Support contact info
   - Async processing (celery)
